# DevLaunch - Collaborative Learning Platform

DevLaunch is a web-based platform where students can learn, build projects, and collaborate in groups. It features AI-powered project planning, real-time group collaboration, task management, and progress tracking.

## 🚀 Features

### User Account
- Register with name, email, and password
- Login with existing credentials
- Edit profile details (phone, address)
- View activity history

### Project Planning
- Generate AI-powered project plans (tech stack, roadmap, tips)
- Use quick templates (Book Club, Todo, Blog, Chat, Portfolio)
- Save projects to your portfolio
- Load previously saved projects
- Star/unstar important projects

### Group Collaboration
- Create public or private groups
- Unique invite code for each group
- Join public groups directly
- Request to join private groups
- Approve/reject join requests

### Task Management
- Add tasks to group
- Assign tasks to members
- Mark tasks as done/undone
- Visual task progress chart

### Shared Resources & Notes
- Share links/resources with group
- Post messages/updates to group
- See who shared each resource

### Learning & Quiz
- Access curated learning resources
- Take AI-generated quiz with instant feedback
- Pass quiz (60%+ score required)

### Progress Tracking
- Auto-tracked 5 milestones
- Visual progress bar
- Progress doughnut chart

### AI Mentor Chat
- Ask questions about your project
- Get coding help and explanations
- Context-aware answers

### Admin Dashboard
- View all registered students
- See average progress across users
- All groups and their members
- Recent activity feed
- Search students by name/email

### Real-time Features
- WebSocket notifications for:
  - New tasks, notes, resources
  - Join requests
  - Member updates
